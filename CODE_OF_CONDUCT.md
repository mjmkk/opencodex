# Contributor Covenant Code of Conduct

## Our Pledge

我们作为维护者和贡献者，承诺让社区对所有人开放、友好与尊重。  
As maintainers and contributors, we pledge to keep this community open, welcoming, and respectful for everyone.

## Our Standards

鼓励行为 / Encouraged behavior:

- 使用建设性、专业、可验证的技术讨论 / Use constructive, professional, and verifiable technical communication.
- 尊重不同经验层级的参与者 / Respect contributors with different levels of experience.
- 对反馈保持开放态度 / Stay open to feedback.
- 为社区整体利益做出清晰沟通 / Communicate clearly for the benefit of the community.

不可接受行为 / Unacceptable behavior:

- 人身攻击、侮辱、骚扰 / Personal attacks, insults, or harassment.
- 恶意披露他人隐私 / Malicious disclosure of private information.
- 任何形式的歧视性言行 / Any discriminatory language or behavior.
- 故意破坏协作秩序的行为 / Deliberate disruption of collaboration.

## Enforcement Responsibilities

维护者有权并有责任对违反本准则的行为采取适当且公平的纠正措施。  
Maintainers have both the right and responsibility to take fair and appropriate corrective actions.

## Scope

本准则适用于本仓库及与项目相关的公开交流空间（Issue、Pull Request、讨论区等）。  
This Code of Conduct applies to this repository and project-related public communication spaces (Issues, Pull Requests, discussions, etc.).

## Enforcement

如遇违规行为，请通过仓库维护渠道联系维护者。
维护者将尽快评估并采取相应措施。
If you encounter violations, contact maintainers through the repository channels.
Maintainers will assess and act as quickly as possible.
